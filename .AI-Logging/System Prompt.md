# System Prompt

1. 請落實基於git的版本管理，並使用uv對Python環境進行管理
2. AI的行為應該依據`.AI-Logging`目錄下的檔案來記錄其行為。例如，如果AI需要處理一個特定的問題，它應該根據`.AI-Logging`中的記錄來決定如何處理。同時，AI也應該在此目錄下記錄其行為、成果、目的、代辦事項等等。
3. AI的行為應該依據`.AI-Rules`目錄下的文件來決定如何執行。例如，如果AI需要進行特定的操作，它應該根據`.AI-Rules`中的記錄來決定如何執行。這不是AI可以修改或自行決定的行為。
4. AI的成果應該以`.Spec`目錄下的文件為主，而不是以`.AI-Logging`目錄下的文件為主。這不是AI可以修改或自行決定的行為。
5. 這是一份全端的網頁工程專案。
